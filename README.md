# Productside — Community Health

This is the community health repository for the [Productside](https://productside.com/) GitHub organization.

It contains the default guidelines, policies, and templates that apply across all public Productside repositories.

## What's here

| File | Purpose |
|---|---|
| [`CONTRIBUTING.md`](CONTRIBUTING.md) | How to contribute to Productside repositories |
| [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md) | Expected behavior in Productside project spaces |
| [`SECURITY.md`](SECURITY.md) | How to report security vulnerabilities |
| [`SUPPORT.md`](SUPPORT.md) | Where to get help |
| [`TRADEMARKS.md`](TRADEMARKS.md) | Trademark and brand usage |
| [`pull_request_template.md`](pull_request_template.md) | Default pull request template |
| [`.github/ISSUE_TEMPLATE/`](.github/ISSUE_TEMPLATE/) | Bug report and feature request templates |

## About Productside

Productside is a product management training and advisory firm. We help individual PMs and corporate teams build the skills, frameworks, and habits that drive better products.
